# ワイン在庫管理システム - Claude AI 指示

## システム概要
このプロジェクトはワイン在庫管理システムです。ワインのコレクション、在庫、評価を効率的に管理するためのアプリケーションです。

## 主な責務

### 1. コード品質の向上
- 既存コードのレビューと改善提案
- ベストプラクティスに従ったコード実装
- テストカバレッジの向上
- ドキュメンテーションの充実

### 2. 機能開発
- 新しいワイン管理機能の実装
- ユーザーインターフェースの改善
- データベーススキーマの最適化
- API エンドポイントの拡張

### 3. バグ修正
- 報告されたバグの調査と修正
- エッジケースの処理改善
- パフォーマンスの最適化
- セキュリティ脆弱性の解決

### 4. ドキュメント作成
- README ファイルの作成・更新
- APIドキュメンテーション
- セットアップガイド
- 使用方法の説明

## 技術スタック

このプロジェクトでは以下のような技術が使用される可能性があります：
- バックエンド: Python, Node.js, その他
- フロントエンド: React, Vue.js, その他
- データベース: PostgreSQL, MongoDB, その他
- テスティング: Jest, pytest, その他

## ワイン管理の主要機能

### 1. ワイン登録
- ワイン名、ぶどう品種、ヴィンテージ年の登録
- 産地、プロデューサー情報
- 購入価格と購入日時の記録

### 2. 在庫管理
- 在庫数の追跡
- 使用/消費記録の管理
- 低在庫アラートの設定

### 3. 評価・レビュー
- テイスティングノートの記録
- 点数/レーティングの管理
- 推奨される飲用時期の記録

### 4. レポート機能
- 在庫レポート生成
- 消費統計の表示
- ワインコレクション分析

- `feature/` - 新機能開発

- `refactor/` - コードリファクタリング

## チェックリスト

コード変更時は以下を確認してください：

- [ ] 既存テストがすべてパスしているか
- [ ] 新しい機能にテストが追加されているか
- [ ] コードがリント/フォーマット要件を満たしているか
- [ ] READMEまたはドキュメントが更新されているか
- [ ] セキュリティの問題がないか
- [ ] パフォーマンスへの悪影響がないか
